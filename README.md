# StayFit with Riddhi

Frontend-only landing page for a fitness and nutrition coaching brand, built with React, Vite, Tailwind CSS v4, and Motion.

## Run locally

Prerequisite: Node.js 20+

1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`

No environment variables or API keys are required.

## Build for production

1. Create a production build with `npm run build`
2. Preview the build locally with `npm run preview`

## Notes before publishing

1. Update the contact and booking destinations if you want the live site to point to your real WhatsApp, Instagram, or email.
2. Replace the externally hosted showcase images with your own local assets if you want the repo to be fully self-contained.
